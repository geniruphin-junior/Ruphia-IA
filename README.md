# MonIA

IA polyvalente capable de :

- Discuter
- Ovrir des apps, fermer des apps
- interface futuriste
- capable de calculer les maths,stats,physique,...
- integration du module pyttsx3

## Lancement

- lancement dans un terminal:

* python main.py

- lancement du serveur:

* python app.py
* ou le launcher
